<h1 align="center">Redo | playing around</h1>

## Idea

I just wanted to implement a redo functionality. The idea is to have a simple mws-server and an in-memory database
system to fastly spin up a basic webserver where I serve the API enabling the CRUD-flow of a basic concept, eg: ITEM.
